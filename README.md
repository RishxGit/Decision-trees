# Decision Trees

A lightweight repository dedicated to learning how Decision Trees work behind the scenes. 

## 🧠 What's Inside
* **`Decision_Trees.ipynb`**: Core notebook where I build and test the tree structure.
* **Custom Split Logic**: Implementation of a `get_best_split` loop to calculate information gain and find the best feature to split on.

## 🛠️ Concepts
Data impurity, Entropy, Information Gain, and Recursive Node Splitting.
